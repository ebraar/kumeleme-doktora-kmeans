# Clustering Comparison Project

## Amaç
Bu projede farklı veri setleri üzerinde K-means ve diğer kümeleme algoritmaları uygulanarak performans karşılaştırması yapılacaktır.

## Kullanılan Yöntemler
- K-means
- (gelecek) Hierarchical Clustering
- (gelecek) DBSCAN

## Veri Setleri
- Dataset 1
- Dataset 2
- Dataset 3

## Değerlendirme Metrikleri
- Silhouette Score
- Davies-Bouldin Index
